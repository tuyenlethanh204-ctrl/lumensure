# Evidence Bundle

This directory contains judge-readable evidence for LumenSure++.

## Files

- `tool-versions.txt`: Local tool versions used for verification.
- `public-input-manifest.json`: Public input order, encoding, domain separator, and mutation coverage.
- `verifier-artifacts.json`: Proof, verifying-key, and public-input artifact hashes generated by `npm run proof:fixture`.
- `local-latest.json`: Local proof/evidence summary without private witness data.
- `mutation-results.json`: Latest local contract mutation test summary.
- `auditor-receipt.json`: Local auditor receipt for proof hash, VK hash, and local final state.
- `testnet-latest.json`: Recorded Stellar testnet deployment and payout evidence from the existing deployment run.
- `testnet-links.md`: Explorer and dashboard links for the recorded deployment.

## Privacy Boundary

Evidence files must not include private witnesses. The private event value, timestamp, nonce, oracle signature, and policyholder witness are generated locally for proof creation and are intentionally omitted from evidence JSON.

## Reproduce

```powershell
npm install
npm run proof:fixture
npm run nargo:test
wsl.exe -d Ubuntu -- bash -lc 'cd /mnt/d/dorahack/stellar/zkinsure && cargo test --manifest-path contracts/insurance/Cargo.toml -- --nocapture'
npm run web:build
```
